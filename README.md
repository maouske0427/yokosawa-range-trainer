# ヨコサワ流ポーカーハンドレンジ・トレーナー

テキサスホールデムのオープンレンジを最短で記憶・定着させるための個人向けトレーニングアプリ。

## 🎯 機能

- **Open Raise クイズ**: 2択（FOLD/RAISE）でオープンレンジを学習
- **Action Decision クイズ**: 3択（FOLD/CALL/3-BET）でアクションを判断
- **Range Painting**: 13x13グリッドを塗りつぶして正解率を確認
- **モード切替**: トーナメント（Anteあり）/ リング（Anteなし）
- **統計表示**: 正解数、正解率、連続正解数などを記録

## 🚀 使い方

### オンラインで使う（推奨）

GitHub Pagesで公開しています：
**https://maouske0427.github.io/personal_dev/yokosawa-range-trainer/**

スマートフォンのブラウザからアクセスして、すぐに使えます！

### ローカルで使う

1. このリポジトリをクローン
2. `index.html` をブラウザで開く

## 📱 スマートフォンで使う

1. 上記のURLをスマートフォンのブラウザ（Safari/Chrome）で開く
2. ホーム画面に追加して、アプリのように使うこともできます

### iPhoneの場合
- Safariで開く → 共有ボタン → 「ホーム画面に追加」

### Androidの場合
- Chromeで開く → メニュー → 「ホーム画面に追加」
